Preview `HTML` files in your browser. Support Chrome, Firefox, Opera, Safari, IE.

## Usage

Three ways to use:
- Command
- Editor Context Menu
- Explorer Context Menu

## Commands

`Ctrl+Shift+P` or `Cmd+Shift+P`

  - View In Browser
  - View In Other Browsers

## Options

`view-in-browser.default`: set default browser
